CSS Selectors:CSS selectors are used to "find" (or select) the HTML elements you want to style

There are five types of CSS Selectors are there, they are:
1.Id Selector (#)
2.Class Selector(.)
3.Tag Selector (Element)
4.Group Selector(div,p,h1,h2)
5.Universal Selector(*)

Example:

Before that create a css style sheet named as Selectors.css(use Link tag)

#abc{
	color:red;  {ID selector}
}

.aaa{
	color:blue; {Class Selector}
}

h1{
	color:red;{Tag selector}
}
div{
	color:green;{Tag selector}
}

div h1{
	color:blue;{Group selector}
}

.parent h1,p,div{
	color: green.{Group Selector}
}

*{
	color:red;{Universal Selector}
}
